# Personal Portfolio Website – HTML5 Semantic Structure & Accessibility

## Project Overview

This project is a multi-page personal portfolio website developed using modern Semantic HTML5 principles and Web Content Accessibility Guidelines (WCAG). The primary goal of this project is to demonstrate the implementation of accessible, well-structured, and SEO-friendly web pages.

The website consists of multiple pages that present personal information, showcase projects, and provide an accessible way for visitors to make contact.

---

## Objectives

- Build a multi-page portfolio website.
- Use Semantic HTML5 elements to create meaningful page structures.
- Follow WCAG accessibility best practices.
- Ensure keyboard navigability throughout the website.
- Implement ARIA labels and attributes to support screen readers.
- Optimize pages using SEO-friendly metadata.
- Create an accessible contact form with proper labels and validation.

---

## Technologies Used

- HTML5
- CSS3
